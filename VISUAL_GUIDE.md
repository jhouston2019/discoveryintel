## Data Flow Visual

### Document Upload → Analysis
```
Upload PDF
    ↓
Store in Supabase Storage
    ↓
Parse text (pdf-parse or OCR)
    ↓
Chunk into segments
    ↓
Generate embeddings (OpenAI)
    ↓
Store vectors (pgvector)
    ↓
Trigger analysis modules
    ↓
    ├─→ Timeline Generator
    ├─→ Contradiction Detector
    ├─→ Impeachment Finder
    ├─→ Evidence Extractor
    ├─→ Deposition Analyzer
    └─→ Strategy Generator
    ↓
Store results in database
    ↓
Display in UI
```

## API Flow Visual

### Authentication Flow
```
Client                  Server                Database
  │                       │                      │
  ├─ POST /auth/signup ──→│                      │
  │                       ├─ Hash password ──────→│
  │                       │←─ Create user ────────┤
  │                       ├─ Generate JWT         │
  │←─ Return user + token─┤                      │
  │                       │                      │
  ├─ POST /auth/login ───→│                      │
  │                       ├─ Verify password ────→│
  │                       │←─ Get user ───────────┤
  │                       ├─ Generate JWT         │
  │←─ Return user + token─┤                      │
```

### Document Upload Flow
```
Client                  Server                Storage/Queue
  │                       │                      │
  ├─ POST /documents/upload →│                   │
  │   (multipart/form-data) │                    │
  │                       ├─ Validate file       │
  │                       ├─ Upload to storage ──→│
  │                       │←─ Confirm upload ─────┤
  │                       ├─ Create DB record     │
  │                       ├─ Queue processing ────→│
  │←─ Return document info─┤                      │
  │                       │                      │
  │                       │  [Background Job]    │
  │                       │←─ Process document ───┤
  │                       ├─ Parse text          │
  │                       ├─ Generate embeddings │
  │                       ├─ Store vectors       │
  │                       └─ Update status       │
```
