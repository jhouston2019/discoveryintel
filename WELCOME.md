## 🔥 Key Features

### Document Processing
Upload PDF, DOCX, or TXT files. The system:
- Extracts text automatically
- Runs OCR on scanned documents
- Chunks text intelligently
- Generates embeddings
- Stores in vector database

### AI Analysis (6 Modules)

1. **Timeline Generator**
   - Extracts dates and events
   - Chronological organization
   - Confidence scoring

2. **Contradiction Detector**
   - Finds conflicts between documents
   - Severity classification
   - Detailed explanations

3. **Impeachment Finder**
   - Witness statement analysis
   - Evidence contradiction matching
   - Strength assessment

4. **Evidence Extractor**
   - Detects admissions
   - Finds inconsistencies
   - Importance ranking

5. **Deposition Analyzer**
   - Key admission extraction
   - Contradiction detection
   - Evasive answer identification

6. **Strategy Generator**
   - Case strengths/weaknesses
   - Leverage points
   - Recommended actions

### Semantic Search
Ask questions in natural language:
- "What contradicts the engineer report?"
- "Evidence of wind damage"
- "Timeline of inspections"

## ⚡ Quick Start

```bash
# 1. Set up Supabase (5 min)
# Create project, run SQL migrations

# 2. Get OpenAI key (2 min)
# Get from platform.openai.com

# 3. Install Redis (3 min)
# brew install redis (Mac)
# apt-get install redis-server (Linux)
# Download for Windows

# 4. Configure (2 min)
cp backend/.env.example backend/.env
cp frontend/.env.example frontend/.env.local
# Edit with your API keys

# 5. Install & Run (3 min)
npm run install:all
cd shared && npm run build && cd ..
npm run dev

# 6. Open browser
# http://localhost:3000
```

**Total: 15 minutes to running app!**
